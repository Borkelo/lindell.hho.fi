Webbdesign

Välkommen till materialet där du lär dig göra hemsidor med hjälp av HTML och CSS. HTML (HyperText Markup Language) är språket som alla webbsidor är kodade i. Sedan man det finnas webbapplikationer (Instagram, Laromedel.fi) som är skapade på andra språk. CSS (Cascading Style Sheets) är språket som används för att formge webbsidorna med färg och form, att placera ut innehåll där det skall vara och att göra webbsidorna responsiva av sig.

Vårt mål för denna helhet är att skapa egna webbsidor från noll med HTML och av att klara av att formge sidorna med CSS.

Innehållsförteckningen är följande

  1. Komma igång med HTML
  2. Stajla med CSS
  3. Responsiva sidor
